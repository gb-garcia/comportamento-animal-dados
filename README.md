# 📊 Análise de Comportamento Alimentar de Bovinos

Projeto de Ciência de Dados aplicado ao agronegócio, com foco na análise de padrões de consumo alimentar a partir de dados de sensores.

---

## 🚀 Problema de negócio

Monitorar o comportamento alimentar de bovinos é essencial para:
- colaborar com eficiência alimentar
- identificar padrões anormais
- melhorar produtividade e sustentabilidade

Este projeto analisa dados de consumo ao longo de 42 dias para identificar padrões e variações individuais.

---

## 📈 Solução

- Tratamento e organização dos dados
- Análise exploratória (EDA)
- Identificação de padrões temporais
- Modelagem estatística (Mixed Effects Model)

---

## 📌 Principais resultados

- Dados simulados não apresentaram padrão temporal relevante  
- Em dados reais, foram observados padrões consistentes de consumo  
- O modelo capturou variações individuais entre os animais  

---

## 🧠 Tecnologias

- Python (Pandas, NumPy)
- Seaborn, Matplotlib
- Statsmodels

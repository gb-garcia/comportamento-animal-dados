## 🚀 Resumo do Projeto

Este projeto analisa o comportamento alimentar de bovinos ao longo de 42 dias utilizando dados de sensores.

### Principais análises:
- Avaliação do consumo ao longo do tempo
- Identificação de padrões individuais
- Aplicação de modelo estatístico (MixedLM)

### Principais resultados:
- Dados simulados não apresentaram padrão temporal relevante
- Em dados reais, foram observados padrões consistentes de consumo
- O modelo foi capaz de capturar variações individuais entre os animais
  
# comportamento-animal-dados
Análise de comportamento alimentar de bovinos com dados de sensores e modelos estatísticos
